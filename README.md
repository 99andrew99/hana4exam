# hana4exam

디지털 하나로4기 평가 문제

## 프론트엔드 기초 평가

### 소스 코드 작성 방법

1. https://github.com/indiflex/hana4exam 에서 Fork 수행
1. 본인 PC에 Fork 받은 repo를 clone
1. FEBasic 폴더 속, 본인 이름의 폴더에 있는 소스코드 작성
1. Fork 받은 본인 github으로 push
1. PR 보내기

### test 요령 실행하여

<본인ID>.test.js 파일을 실행하여 통과되면 됨.
